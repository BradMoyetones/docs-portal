Bienvenido al centro de documentación de los proyectos internos del Parque Jaime Duque. Aquí encontrarás información detallada sobre cada uno de los sistemas que facilitan la operación diaria del parque.

## Nuestros Sistemas

Explora la documentación de cada proyecto para conocer sus funcionalidades, guías de uso y características técnicas.

- **Almuerzos:** Sistema de registro diario de almuerzos para colaboradores del parque.
- **Ecotrueque:** Gestión de residuos aprovechables con sistema de puntos y peso disponible para canjear.
- **Hoteles:** Registro de hoteles aliados para ofrecer cortesías y descuentos a huéspedes.
- **Porteros:** Programación semanal de porteros con visualización individual y reportes gráficos.
- **Registro de Visitantes:** Sistema de registro y analítica de visitantes por región, país y demografía.
- **Carnets:** Gestión de carnets de empleados con búsqueda por cédula y administración centralizada.
- **Biblioteca:** Sistema de gestión para Biblioteca Nuevos Horizontes con catálogo y préstamos.

## Características Generales

<Callout className="mt-6 mb-6 border-blue-600 bg-blue-50 dark:border-blue-900 dark:bg-blue-950 [&_code]:bg-blue-100 dark:[&_code]:bg-blue-900">
  **Importante:** Recuerda que para el desarrollo de cada una de estas apps se utilizaron 
  diferentes [Bibliotecas de código abierto](/docs/libraries) para mayor rapidez en el desarrollo.
</Callout>

Todos nuestros sistemas están diseñados con:

- **Interfaz intuitiva** - Fácil de usar para todos los colaboradores
- **Seguridad** - Protección de datos y acceso controlado por roles
- **Reportes** - Generación de estadísticas y análisis en tiempo real
- **Responsive** - Accesible desde cualquier dispositivo
- **Integración** - Sistemas conectados para mayor eficiencia (como Ecotrueque con Almuerzos)
