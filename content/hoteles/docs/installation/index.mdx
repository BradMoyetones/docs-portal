## Requisitos Previos

Antes de comenzar con la instalación, asegúrate de tener instalado lo siguiente:

### Para la API (Backend)

- **PHP 8.0 o superior** con las siguientes extensiones:
  - PDO
  - pdo_mysql
  - mbstring
  - openssl
- **Composer** (gestor de dependencias de PHP)
- **Apache** o **Nginx** como servidor web
- **MySQL 5.7 o superior** o **MariaDB 10.3 o superior**

### Para el Frontend

- **Node.js 18.0 o superior**
- **npm** o **yarn** (gestor de paquetes de Node.js)

## Orden de Instalación Recomendado

Para una instalación completa del sistema, se recomienda seguir este orden:

1. **Instalar y configurar la API** primero, ya que el frontend depende de ella
2. **Instalar y configurar el Frontend** una vez que la API esté funcionando

<Callout>
  **Nota:** Ambos proyectos están en repositorios de GitHub separados. Asegúrate de tener acceso a ambos repositorios antes de comenzar.
</Callout>
