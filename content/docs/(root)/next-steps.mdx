- **`types.mdx`** - Todos los tipos e interfaces principales (User, Hotel, Guest, CourtesyRecord, etc.)

- **`components/hotel-card.mdx`** - Tarjeta de hotel con ejemplos de uso

## Servicios y API

- **`services.mdx`** - Cómo funcionan los servicios de API, interceptores de Axios, manejo de errores
- **`api-integration.mdx`** - Patrones para consumir la API, ejemplos de peticiones


## Guías Prácticas

- **`guides/adding-new-page.mdx`** - Cómo agregar una nueva página/módulo

- **`guides/testing.mdx`** - Si tienen tests, cómo escribirlos


## Utilidades
